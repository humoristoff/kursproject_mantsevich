# Классификатор культурных орнаментов

Десктопное приложение для распознавания культурных орнаментов по изображениям на основе нейросетевых методов (ViT, датасет JONES-19).

## Сборка exe
Так как файл `desktop_app.exe` имеет большой объем (~500 МБ), он выложен отдельно:

- [Скачать приложение (Google Drive)](https://drive.google.com/drive/folders/1wnKgbufr9MPyRxQ3Ei86VdmXwDPp9DFS)

## Краткое руководство пользователя

### 1. Распаковка и установка
1. Скачайте архив с приложением по ссылке выше.
2. Распакуйте архив в любую папку (например, `C:\OrnamentApp` или на рабочий стол).
3. Убедитесь, что в распакованной папке находятся:
   - `desktop_app.exe` – исполняемый файл приложения
   - `vit_jones19_model` – папка с обученной моделью
   - `vit_jones19_processor` – папка с процессором изображений

**Важно:** папки `vit_jones19_model` и `vit_jones19_processor` должны лежать **рядом** с `desktop_app.exe`. Без них приложение не запустится.

### 2. Запуск приложения
Дважды кликните по файлу `desktop_app.exe`. Никаких дополнительных библиотек или Python устанавливать не требуется. Приложение работает полностью автономно, без подключения к интернету.

### 3. Работа с приложением
- **Загрузить изображение:** нажмите кнопку «Выбрать изображение», выберите файл в формате JPEG, PNG или BMP.
- **Результат классификации:** после загрузки приложение автоматически определит три наиболее вероятных культурных стиля с указанием процента уверенности.
- **Экспорт истории:** кнопка «Экспорт CSV» открывает файл `classification_history.csv`, в котором сохраняются все выполненные классификации.

### 4. Тестовые изображения
Примеры изображений для тестирования можно найти в папке `src/test`. Рекомендуются изображения с чёткими, крупными орнаментами.
